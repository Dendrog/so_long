# 42 So Long

## 개요
C 언어와 MiniLibX를 사용하여 **2D 게임**을 구현하는 프로젝트입니다.  
플레이어가 맵을 탐험하며 아이템을 수집하고, 출구를 찾아 탈출하는 게임 로직을 구현합니다.

## 기능
- 2D 맵 기반 게임 구현
- 플레이어 이동, 충돌 처리 및 아이템 수집
- 맵 종료 조건 구현 (모든 아이템 수집 후 출구 도달)
- 키 입력 처리 및 화면 갱신

## 사용법
```bash
# 컴파일
make

# 실행 예시
./so_long map.ber
```
